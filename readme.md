## **Что это такое?**
Это **полноценный веб-мессенджер**, похожий на Telegram Web или WhatsApp Web, который работает прямо в браузере. Не нужно ничего устанавливать — достаточно зайти на сайт с любого устройства (компьютер, планшет, телефон) и начать общение.

## **Ключевые возможности:**

### 1. **Регистрация и вход по телефону/логину**
- Быстрая регистрация с именем, фамилией и телефоном
- Вход по номеру телефона 
- Безопасное хранение паролей (зашифрованы)

### 2. **Личные чаты 1 на 1**
- Создание приватных бесед с любым пользователем
- Мгновенная отправка текстовых сообщений
- История переписки всегда доступна

### 3. **Поиск пользователей**
- Умный поиск по имени, фамилии, телефону или нику
- Автоматическое создание чата с найденным человеком
- Можно начать общение с любым зарегистрированным пользователем

### 4. **Онлайн-статусы в реальном времени**
- Видно, кто сейчас онлайн (зеленый индикатор)
- Точное время последней активности ("был(а) 5 минут назад")
- Автоматическое обновление статусов

### 5. **Удобный интерфейс**
- Боковая панель со списком чатов
- Основное окно переписки
- Адаптивный дизайн (работает на компьютерах и телефонах)
- Цветные аватары для каждого пользователя

### 6. **Уведомления о прочтении**
- Видно, когда сообщение доставлено собеседнику
- Индикатор прочтения (+/-)
- Подробная информация о времени отправки и прочтения

### 7. **Управление чатами**
- Полное удаление чатов для всех участников
- Автоматическая сортировка чатов (последние активные сверху)

##  **Технологии:**

### **Backend (серверная часть):**
- **FastAPI** — современный высокопроизводительный фреймворк для API
- **PostgreSQL** — надежная база данных для хранения сообщений и пользователей
- **SQLAlchemy** — профессиональная работа с базой данных
- **JWT-токены** — безопасная авторизация
- **WebSockets-ready** — готовность к мгновенным уведомлениям

### **Frontend (интерфейс):**
- **Чистый HTML/CSS/JavaScript** — работает быстро в любом браузере
- **Адаптивный дизайн** — подстраивается под размер экрана
- **Асинхронные запросы** — плавная работа без перезагрузки страницы

### **Безопасность:**
- Шифрование паролей
- Защищенные HTTPS-соединения
- Защита от несанкционированного доступа
- Безопасное хранение сессий
