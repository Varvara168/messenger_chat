## 🎯 **Что это такое?**
Это **полнофункциональный веб-мессенджер**, который работает прямо в браузере — как Telegram Web или WhatsApp Web, но с вашим собственным дизайном и функциями. Не нужно ничего устанавливать: заходите на сайт с компьютера, планшета или телефона и сразу начинаете общаться.

## ✨ **Что умеет приложение (главные функции):**

### 1. **📝 Быстрая регистрация и вход**
- Регистрация за 1 минуту: имя, фамилия, телефон, пароль
- Вход по номеру телефона или короткому имени (например, @Анна)
- Безопасное хранение данных

### 2. **💬 Личные сообщения 1 на 1**
- Создаете чат с любым пользователем
- Пишете сообщения — они приходят мгновенно
- Вся история переписки сохраняется
- Можно удалить чат полностью (для обоих участников)

### 3. **🔍 Умный поиск людей**
- Ищете по имени, фамилии, телефону
- Нашли человека — одним кликом создаете с ним чат
- Если чат уже был — он просто открывается

### 4. **🟢 Видно, кто онлайн**
- Зеленый значок — человек сейчас в сети
- Если не в сети — показывается, когда был последний раз
- Статусы обновляются автоматически

### 5. **✓ Подтверждение прочтения**
- ## - — сообщение доставлено
- ## + — сообщение прочитано
- Можно посмотреть точное время отправки и прочтения

### 6. **🎨 Удобный интерфейс**
- Слева — список всех ваших чатов
- Справа — переписка с выбранным человеком
- Цветные кружочки с буквами вместо фото (у каждого свой цвет)
- Работает на любом устройстве
